# Shadow Techniques for Brawl Stars Typography

## Drop Shadow Presets

### Hard Comic Shadow
- Distance: 6px X, 6px Y
- Spread: 100%
- Size: 0px
- Color: #000000 100%
- Effect: Crisp, cartoon-like separation

### Soft Ambient Shadow
- Distance: 4px X, 8px Y
- Size: 8px
- Color: #1a1a2e 70%
- Effect: Natural depth, matches top-left lighting

### Long Extrusion Shadow
- Distance: 25px X, 25px Y
- Spread: 100%
- Size: 5px
- Color: #000000 50%
- Effect: 3D block extrusion

### Shadow Color Rules
- Never use pure #000 black — sample the darkest color in the composition
- For light/bright thumbnails: use #1a1a2e (dark navy)
- For dark thumbnails: use a darker version of the background color
- Match the shadow angle to the composition's light source

### Offset Decisions by Composition
| Light Source | Shadow X | Shadow Y |
|-------------|----------|----------|
| Top-left | +6px | +6px |
| Top-right | -6px | +6px |
| Bottom-left | +6px | -6px |
| Bottom-right | -6px | -6px |
| Center glow | 0px | +8px (down only) |
