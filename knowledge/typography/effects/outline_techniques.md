# Outline Techniques for Brawl Stars Typography

## Standard Outside Outline
- Layer style → Stroke → Outside
- Size: 6-10px at 1080p
- Color: #1a1a2e (dark navy — matches Brawl Stars UI)
- Blend mode: Normal, 100%

## Inline Outline
- Layer style → Stroke → Inside
- Size: 3-4px
- Color: White or highlight color
- Effect: Creates a highlight rim inside the letter
- Best for: Dark colored text on dark backgrounds

## Multi-Layer Outline
- Layer 1: 10px dark outline (standard)
- Layer 2: 6px mid-tone outline (inside Layer 1, thinner)
- Layer 3: 3px highlight outline (inside Layer 2)
- Result: Three-step outline creating a premium bevel effect

## Color Separation Outline
- Three text layers:
  1. Bottom: Dark color + thick outline, moved +4px right/down
  2. Middle: Mid color + medium outline, moved +2px
  3. Top: Light/white fill + thin outline
- Result: 3D extrusion effect without actual 3D
