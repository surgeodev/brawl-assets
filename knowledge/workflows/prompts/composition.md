# Prompt: Design Composition

## Objective
Arrange the selected assets into a composition following the blueprint.

## Input
- Blueprint (composition type, depth layers, focal point)
- Selected assets (from asset_search.md)

## Composition Rules

### Rule 1: Follow the Grid
Place elements according to the composition type defined in the blueprint:
- Rule of thirds: Align main subject to left or right vertical line
- Diagonal: Place main subject at one end, flow toward the opposite
- Centered: Main subject in center, but offset secondary elements
- Never center everything — vary the horizontal and vertical positions

### Rule 2: Create Depth with Sizing
- Foreground elements: Large, partially cropped by frame edge
- Midground elements (main subject): Full visibility, 40-60% of frame
- Background elements: Full frame, lower opacity or darker
- Overlay elements: Same layer as text, on top of everything

### Rule 3: Scale Relationships
```
Main subject:      100% (reference)
Foreground element: 60-80% of frame (partial)
Secondary subject:  40-60% of main
Background element: covers 100% of frame
Text:               20-35% of frame height
```

### Rule 4: Visual Flow
Create a path for the eye to follow:
- Z-pattern: Top-left → Top-right → Bottom-left → Bottom-right
- Diagonal: One corner to the opposite
- Circular: Around the focal point
- The flow must start at the focal point and end at the text or CTA

### Rule 5: Negative Space
- Reserve 20-30% of the frame for negative space
- Place text in negative space
- Negative space should form a "shape" — not just leftover area

### Rule 6: Visual Weight Balance
- Left side weight vs. right side weight should be 60:40 or 40:60 — never 50:50
- The side with the subject should have more weight
- Balance with dark/light contrast, not just element size

## Composition Anti-Patterns
- ❌ Subject dead center with text below
- ❌ All elements crowded in the middle third
- ❌ No clear path for the eye
- ❌ Visual weight equal on both sides
- ❌ Background more interesting than the subject

## Output
A description of the composed scene with exact element positions, sizes, and depth layer assignments.
