# Prompt: Final Quality Control

## Objective
Perform the final quality check before a thumbnail is considered complete.

## The 5-Second Test

Show the thumbnail to someone for 5 seconds, then take it away.
Ask them:
1. What was the video about?
2. Who was the main subject?
3. How did it make you feel?

If all three answers match the blueprint's core_message, main_subject, and emotion, the thumbnail passes.

## The 25% Readability Test

1. Resize the thumbnail to 480x270px (25% of 1920x1080)
2. View at arm's length
3. Check:
   - Is the main subject identifiable?
   - Is the text perfectly readable?
   - Is the focal point clear?
4. If any of these fail, the thumbnail needs adjustment

## The Thumbnail-Only Test

Show ONLY the thumbnail (no title, no channel name, no context).
Can the viewer guess:
- Which game this is? (Brawl Stars visual identity must be clear)
- What the video is about? (core_message must be communicated visually)
- Who the main subject is? (main brawler must be identifiable)

## The Competition Test

Place the thumbnail next to 5 other Brawl Stars thumbnails in a grid.
Does it:
- Stand out from the group?
- Look professionally designed?
- Make you want to click it?
- Look like it belongs to the same genre (Brawl Stars content)?

## Technical Checklist

```
□ Resolution: 1920x1080px (16:9)
□ Format: PNG or JPEG (PNG for high quality, JPEG for smaller file)
□ File size: Under 2MB for YouTube thumbnails
□ No compression artifacts visible
□ Aspect ratio is exactly 16:9
□ No unintentional borders or margins
□ All layers are flattened to a single image
□ Color profile: sRGB
```

## Export Settings

```
For YouTube:
  Resolution: 1920x1080px (16:9)
  Format: PNG (preferred) or JPEG (quality 95%+)
  Max file size: 2MB
  
For Preview / Portfolio:
  Resolution: 1920x1080px
  Format: WebP (quality 90)
  Max file size: 500KB
```

## Final Sign-Off

The thumbnail is complete when:
1. All 8 review sections pass
2. The 5-second test passes
3. The 25% readability test passes
4. The thumbnail-only test passes
5. The competition test passes
6. Technical checklist is clear
7. File is exported in correct format and size
