# Thumbnail Analysis Template

Use this template to analyze any Brawl Stars thumbnail example.

```yaml
thumbnail_analysis:
  source: "Reference image filename or source URL"
  
  # VISUAL OBJECTIVE
  visual_objective: "What is this thumbnail trying to communicate?"
  
  # MAIN SUBJECT
  main_subject:
    brawler: "Name"
    skin: "Skin"
    pose: "Description"
    facing: "Direction"
    scale: "% of frame"
  
  # COMPOSITION
  composition:
    type: "Rule of thirds / centered / diagonal / dynamic"
    focal_point: "Where does the eye land?"
    visual_flow: "How does the eye move across the composition?"
  
  # DEPTH
  depth_layers:
    - layer: "Foreground"
      elements: ["List of elements"]
      purpose: "Why are they there?"
    - layer: "Midground"
      elements: ["Main subject area"]
    - layer: "Background"
      elements: ["Environment"]
    - layer: "Overlay"
      elements: ["Text, particles"]
  
  # LIGHTING
  lighting:
    primary_direction: "Where from?"
    secondary_direction: "Fill light?"
    shadow_consistency: "Do all shadows match?"
  
  # COLOR
  color_palette:
    dominant: "Hex"
    accent: "Hex"
    background: "Hex"
    white_space: "Hex"
  
  # TYPOGRAPHY
  typography:
    font: "Name or closest match"
    treatment:
      fill: "Solid / gradient / empty"
      outline: "Thickness, color"
      shadow: "Offset, blur"
    placement: "Where on the grid?"
    hierarchy: "Which element reads first?"
  
  # DESIGN DECISIONS
  what_makes_it_professional:
    - "Specific intentional choice 1"
    - "Specific intentional choice 2"
    
  what_makes_it_NOT_generic_ai:
    - "Reason 1"
    - "Reason 2"
  
  # IMPROVEMENT (if any)
  weaknesses:
    - "What could be better?"
  
  # REUSABLE TECHNIQUE
  technique_to_steal:
    "What one technique from this thumbnail can be reused?"
```
